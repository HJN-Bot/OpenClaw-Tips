## Minimal KPI Set (recommended)

### 1) Recovery / Energy
- **Sleep hours (avg):** target ≥ 7.0h, redline < 6.0h for 2 days
- **Morning energy score (1–5):** target ≥ 3.5, redline ≤ 2 for 2 days

### 2) Training Consistency
- **Sessions completed / planned:** target ≥ 85%, redline < 60% weekly
- **Low-intensity movement days:** target ≥ 5/week, redline < 3/week

### 3) Emotional Stability
- **Mood check-ins completed:** target ≥ 5/week, redline < 3/week
- **High-stress days (self-rated 4–5/5):** target ≤ 2/week, redline ≥ 4/week

### 4) Execution Hygiene
- **Top-3 priorities done:** target ≥ 70% daily, redline < 40% for 3 days
- **Critical reply latency:** target same-day, redline > 24h on important items

## Status Rules

- **On track:** at/above target
- **Watch:** between target and redline
- **Off track:** at/below redline

When **Off track** → define one concrete fix with owner + date.
