# 内容链接丝滑采集管线（XHS/微信/公众号）

## 目标
把链接采集做成双通道：
- 通道A：web_fetch 快速抓取
- 通道B：浏览器接管抓取（Relay）
失败时自动降级到人工转存。

## 流程
1. URL平台识别（xhs/wechat/mp/other）
2. A通道抓取：成功则解析正文与元信息
3. A失败或低完整度 -> B通道
4. B失败 -> 人工转存入口（粘贴正文）
5. 标准化输出：title/author/date/platform/body/tags/source_url
6. 回写：Task Desc + 素材库索引

## 质量指标
- 抓取成功率（按平台）
- 正文完整率
- 平均耗时
- 失败可恢复率

## 安全与合规
- 尊重平台条款
- 不绕过认证保护
- 仅处理用户授权/公开可访问内容
