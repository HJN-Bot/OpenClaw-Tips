# 8D Optimization Skill (Shared)

This is the shared optimization framework used by Andrew/Rex/Lulu/Alex.

## Core idea
The 8 directions are **not one-off tasks**; they are a reusable diagnostic and optimization operating system.

## Use on every new task
1. Classify task type (Delivery vs Capability)
2. Score 8D quickly
3. Select top-3 improvements
4. Apply approval gate + risk gate
5. Write back evidence

## Why this matters
- Converts local fixes into long-term capability
- Keeps recommendations consistent across projects
- Prevents overbuilding and fragmented execution
